Cette règle permet de traquer l'utilisation d'une classe logger en fonction de la méthode utilisée (info, warn, error ou debug).

Noncompliant Code Example

  gameState.print(System.out);
  System.out.println("The board is full. You lost!");

Compliant Solution

    LOGGER.info("The board is full. You lost!");
    // other invocation
    log.info("The board is full. You lost!");
